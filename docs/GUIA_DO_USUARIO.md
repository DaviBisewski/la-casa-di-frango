## Começar

### Instalar no iPad

1. Abra o app no **Safari** (navegador padrão do iPad)
2. Toque no botão **Compartilhar** (↑ na barra inferior)
3. Role para baixo e selecione **"Adicionar à Tela de Início"**
4. Digite o nome: **"Frango"** ou **"La Casa Di Frango"**
5. Toque em **"Adicionar"**
6. O app agora aparece na tela inicial como um ícone

O app funciona 100% offline — você pode usar sem internet!

### Instalar no Celular (Android)

1. Abra o app no **Chrome**
2. Toque no menu (⋮) no canto superior direito
3. Toque em **"Instalar app"**
4. Confirme
5. O app aparece na tela inicial

### Instalar no Celular (iPhone/iOS)

1. Abra o app no **Safari**
2. Toque em **"Compartilhar"** (↑)
3. Selecione **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"**

### Usar no Computador

1. Abra em qualquer navegador: Chrome, Safari, Firefox, Edge
2. Marque como favorito (Ctrl+D) para acesso rápido
3. Funciona normalmente offline

## Usando o Sistema

### 1️⃣ Iniciando o Dia (Novo Expediente)

#### Passo 1: Clicar em "Começar Expediente"

No topo da tela inicial, clique no botão grande verde **"Começar Expediente"**

#### Passo 2: Inserir Estoque Inicial

Você verá campos para:
- **Frango Com Recheio** — quantidade que começou o dia
- **Frango Sem Recheio** — quantidade inicial
- **Meio Frango** — quantidade inicial

Digite os números que tem em estoque agora.

**Se for domingo**, você também verá:
- **Maionese P (R$10)** — quantidade
- **Maionese G (R$15)** — quantidade
- **Costela** — quantidade

#### Passo 3: Confirmar

Clique no botão **"Começar Expediente"**

Se tudo está certo, o app leva você automaticamente para o **Dashboard**.

### 2️⃣ Dashboard (Expediente Ativo)

A tela mostra:

**Topo**: 
- Ícone de estoque e título "Estoque"
- Status: verde **"Ativo"** ou cinza **"Encerrado"**

**Meio da tela**:
- Cada produto em um card separado
- Mostra: Quantidade original | Encomendado | Vendido | **Disponível**
- A quantidade em **negrito** é o que você pode vender agora

**Se for domingo**:
- Filtro superior: **"Frangos"** | **"Maioneses"** | **"Costela"**
- Clique para alternar entre categorias

**Botões de ação** (se expediente ativo):
- 🛍️ **ENCOMENDA** — registrar novo pedido
- 💰 **VENDA** — registrar venda balcão

**Abaixo**:
- Lista de todas as encomendas do dia
- Para cada uma: nome, produtos, botão para marcar como retirado

**Rodapé**:
- Botão vermelho **"Encerrar Expediente"** (se ativo)

### 3️⃣ Registrando uma Encomenda

#### Passo 1: Clicar em "ENCOMENDA"

No Dashboard, clique no botão azul **"ENCOMENDA"**

#### Passo 2: Preencher Dados do Cliente

- **Nome**: (obrigatório) — nome da pessoa que encomendou
- **Telefone**: (opcional) — número para contato

#### Passo 3: Selecionar Produtos

Você verá cards com cada produto:
- **Frango Com Recheio**
- **Frango Sem Recheio**
- **Meio Frango**
- (E se for domingo: **Maionese P**, **Maionese G**, **Costela**)

Para cada um:
1. Clique no card
2. Use os botões **-** e **+** para ajustar a quantidade
3. O sistema mostra o máximo disponível (não deixa vender mais)

#### Passo 4: Confirmar

Clique em **"Confirmar Encomenda"**

A encomenda aparece na lista abaixo. Você vê um toast verde: **"✓ Encomenda registrada!"**

### 4️⃣ Registrando uma Venda Rápida

Para vendas de balcão (cliente anônimo):

#### Passo 1: Clicar em "VENDA"

No Dashboard, clique no botão **"VENDA"**

#### Passo 2: Selecionar Produtos

Mesmo que encomenda:
- Cards com cada produto
- Use **-** e **+** para quantidade
- Não precisa preencher nome/telefone

#### Passo 3: Confirmar

Clique em **"Confirmar Venda"**

A venda é registrada (não aparece na lista — apenas desconta do disponível)

### 5️⃣ Marcando Encomenda como Retirada

Quando o cliente busca a encomenda:

#### No Dashboard:

1. Role até **"Lista de Encomendas"** (abaixo dos cards)
2. Encontre a encomenda (procure pelo nome se tiver muitas)
3. À direita de cada encomenda, tem um **botão verde redondo** (✓)
4. Clique nele

A encomenda fica **marcada como retirada** (muda de cor ou fica "apagada")

#### Procurar Encomenda:

Se tem muitas, use a **barra de busca**:
- Digite nome ou telefone
- Lista filtra em tempo real

### 6️⃣ Consultando o Histórico

#### Passo 1: Voltar para Tela Inicial

Clique no ícone **"Casa"** (home) ou volte navegando

#### Passo 2: Ver Lista de Expedientes

Você vê:
- Expediente de hoje (se houver) — com status verde **"Ativo"**
- Expedientes anteriores — em ordem de data (mais recente primeiro)

#### Passo 3: Filtrar por Data (Opcional)

No topo, tem um ícone de **calendário**.

Clique nele para:
- Selecionar uma data específica
- Ver expedientes apenas dessa data
- Limpar filtro com **"X"**

#### Passo 4: Visualizar Expediente Passado

Clique em qualquer expediente da lista:
- Se é expediente hoje e está **ativo**: abre Dashboard
- Se é **encerrado**: mostra tela de visualização (apenas para consulta)

### 7️⃣ Encerrando o Dia

#### Passo 1: Voltar ao Dashboard

Se não estiver, clique no expediente de hoje na tela inicial

#### Passo 2: Rolar até o Fim

Role a página até ver o **botão vermelho "Encerrar Expediente"**

#### Passo 3: Confirmar

Clique no botão:
- Uma mensagem aparece: **"Tem certeza? Não será possível desfazer"**
- Clique em **"Confirmar"**

#### O que Acontece:

- Expediente é **encerrado** (status muda para cinza)
- Não pode mais adicionar encomendas/vendas
- Expediente move para o **histórico**
- Próxima vez que você criar um novo expediente, pode começar do zero

## Backup e Dados

### Fazer Backup Manual

**Para que**: Guardar cópia segura de todos os dados em um arquivo

#### Passo 1: Ir para Configurações

Na tela inicial, clique no ícone **⚙️ (engrenagem)**

#### Passo 2: Exportar Dados

Procure a seção **"Backup Local"**

Clique em **"📥 Exportar Dados"**

#### Passo 3: Arquivo é Criado

Um arquivo chamado `frango_backup_2026-05-11.json` é baixado (ou pode salvar em nuvem)

**Onde fica**: 
- **iPad**: Documentos → Downloads
- **Celular Android**: Downloads
- **Computador**: Pasta Downloads

#### Usar Depois:

Guarde esse arquivo em lugar seguro:
- Nuvem (Google Drive, OneDrive, iCloud)
- Email (envie para você mesmo)
- Pendrive

### Restaurar de Backup

Se seus dados sumiram ou você está usando em outro dispositivo:

#### Passo 1: Ir para Configurações

Clique em **⚙️ (engrenagem)**

#### Passo 2: Importar Dados

Na seção **"Backup Local"**, clique em **"📤 Importar Dados"**

#### Passo 3: Selecionar Arquivo

O sistema pede para você escolher um arquivo `.json`

Selecione o backup que salvou anteriormente

#### Passo 4: Confirmar

Sistema importa os dados. Você vê:
- Toast verde: **"✓ 15 expedientes importados com sucesso"**
- Histórico agora tem os dados antigos

**Importante**: Importação **faz merge** — não deleta dados que você já tinha. Se tiver 10 expedientes e importar 15, fica com 25 (sem duplicar).

## Dúvidas Frequentes

### ❓ Perdi meus dados! Como recupero?

**Resposta**: 

1. Se fez backup antes: [Restaurar de Backup](#restaurar-de-backup)
2. Se não fez: Dados podem estar em `localStorage` (backup automático)
   - Vá para Configurações → Status de Sincronização
   - Verifique "Último backup"

3. Se sincronizou com nuvem (Supabase ativo):
   - Vá para Configurações
   - Clique em "Sincronizar com Nuvem"
   - Dados podem ser recuperados de lá

### ❓ O app não funciona sem internet?

**Resposta**: Funciona 100% offline! 

- Crie expediente
- Registre encomendas
- Faça vendas
- Marque retiradas

**Tudo funciona sem conexão.** Quando voltar online, sincroniza automaticamente com nuvem.

### ❓ Posso usar em múltiplos dispositivos ao mesmo tempo?

**Resposta**: Sim, mas com cuidado:

- iPad em uma tela registrando encomendas
- Celular em outra registrando vendas

Os dados sincronizam entre dispositivos (se online). Mas é melhor:
1. **Usar um dispositivo por vez** para evitar conflitos
2. Sempre **sincronizar** antes de trocar de aparelho

### ❓ Como sincronizar entre dois dispositivos?

**Resposta**:

1. **Dispositivo A** (iPad):
   - Use o app normalmente
   - Sistema sincroniza automaticamente com nuvem quando online

2. **Dispositivo B** (Celular):
   - Abra o app
   - Vá para Configurações → "Sincronizar com Nuvem"
   - Clique em "Baixar dados da nuvem"
   - Dispositivo B agora tem os mesmos dados que A

### ❓ Posso reabrir um expediente que encerrei?

**Resposta**: Não. Uma vez encerrado, não pode modificar.

Se precisa adicionar algo que esqueceu:
1. Faça backup (para segurança)
2. Crie novo expediente para hoje
3. Registre o que faltou nele

### ❓ O que significa aquele ícone de nuvem na tela?

**Resposta**: 

- ☁️ **Verde** = Conectado à internet, dados sincronizando
- ☁️ **Cinza** = Sem internet, dados salvando localmente (sincronizam depois)
- ☁️ com **!** = Erro ao sincronizar (verifique conexão)

### ❓ Quantos expedientes pode ter no histórico?

**Resposta**: Ilimitado! O app funciona com 100, 1.000, ou mais expedientes.

Se ficar lento:
- Faça backup
- Exporte os dados
- Limpe dados antigos (apenas se tiver backup)

### ❓ Esqueci a senha/login?

**Resposta**: Não há senha ou login!

O app abre direto. Se quiser usar em outro iPad/celular:
1. Abra o app
2. Sincronize com nuvem
3. Dados aparecem automaticamente

### ❓ Posso editar uma encomenda/venda depois de registrar?

**Resposta**: Não. Sistema não permite editar.

Se errou:
1. Anote qual foi a mudança
2. Registre como nova encomenda/venda de correção
3. No fim do dia, corrija no relatório manual

### ❓ O arquivo JSON de backup está corrompido. E agora?

**Resposta**: 

1. Verifique se o arquivo é um `.json` válido (abra no editor de texto)
2. Se está ilegível, tente outro backup mais antigo
3. Se não tem backup, dados podem ser recuperáveis do servidor

### ❓ Como Funciona o Modo Domingo?

**Resposta**:

- **Detectado automaticamente** no domingo (sistema vê data/dia)
- Ao criar expediente no domingo: campos extras aparecem
- Produtos especiais: Maionese P, Maionese G, Costela
- Se criar durante semana: produtos dominicais não aparecem

### ❓ Posso gerar relatório de vendas?

**Resposta**: Dados estão no histórico de expedientes:

1. Cada expediente mostra:
   - Data e hora
   - Estoque inicial
   - Encomendas registradas
   - Vendas realizadas
   - Itens vendidos

2. Para relatório completo:
   - Exporte como JSON
   - Abra em Excel ou Google Sheets
   - Faça análise customizada

### ❓ Preciso de suporte/tem bug?

**Resposta**:

1. Limpe cache do navegador (Ctrl+Shift+Delete)
2. Feche o app e abra novamente
3. Se persiste, note:
   - Que ação estava fazendo
   - Que erro apareceu
   - Que navegador/dispositivo usa
4. Entre em contato com suporte

## 💡 Dicas

✅ **Faça backup regularmente** — use botão de exportação 1x por semana

✅ **Use nomes claros para clientes** — facilita busca depois
